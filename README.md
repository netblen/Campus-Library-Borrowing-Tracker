# Assessment4 - Library Management System
A SwiftUI and Core Data application for tracking library inventory, managing members, and recording book loans.

## Setup Instructions
Clone the Repository: git clone https://github.com/netblen/Assessment4.git

Open Project: Launch Assessment4.xcodeproj in Xcode 16.2+.

Run: Select an iPhone 16 or 17 simulator (iOS 18.2) and press Cmd + R or the play button to start.
